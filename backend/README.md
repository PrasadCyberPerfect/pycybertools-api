# PyCyberTools — Backend API
Deploy this folder on Render.com (free)

## Deploy Steps
1. Push this folder to GitHub repo: `pycybertools-api`
2. Go to render.com → New Web Service → Connect repo
3. Settings auto-detected from render.yaml
4. Get your URL: `https://pycybertools-api.onrender.com`
5. Update `const API = "..."` in frontend/index.html

## Local Testing
```
pip install -r requirements.txt
python app.py
```
